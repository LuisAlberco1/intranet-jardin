# intranet-jardin
Descripción del proyecto:
La Intranet Jardín Infantil es una plataforma web desarrollada para mejorar la gestión y comunicación interna del establecimiento. El sistema permite administrar información importante del jardín mediante distintos módulos, facilitando el trabajo administrativo y la organización de actividades.

El objetivo principal es centralizar procesos internos mediante una aplicación desarrollada con React y TypeScript, incorporando autenticación de usuarios, navegación entre módulos y persistencia de datos.

Tecnologías utilizadas:
- React
- TypeScript
- Vite
- React Router
- LocalStorage
- CSS
  
Instalación y ejecución local:

Clonar el repositorio:
git clone URL_DEL_REPOSITORIO

Ingresar a la carpeta:
cd intranet-jardin

Instalar dependencias:
npm install

Ejecutar el proyecto:
npm run dev

Módulos del sistema:
Inicio de sesión:
- Inicio de sesión mediante credenciales simuladas.
- Persistencia de sesión mediante localStorage.
- Gestión global de autenticación mediante Context API.
  
Gestión de comunicados:
- Crear comunicados.
- Editar comunicados.
- Eliminar comunicados.
- Listado dinámico y filtros.
  
Gestión de reuniones:
- Crear reuniones de apoderados.
- Editar reuniones.
- Eliminar reuniones.
- Filtrar reuniones registradas.

Calendario de actividades:
- Registrar actividades.
- Editar actividades.
- Eliminar actividades.
- Mostrar actividades programadas.

Panel administrativo:
- Acceso central a módulos internos.
- Navegación protegida mediante autenticación.
